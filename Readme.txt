FilmStartPage - Readme


NOTE: You need all the files in the folder to run the application.
      Do not remove the application or files from the folder.


To run the application, click on the FilmStartPage.html file.
This will display the current FilmScreen.jpeg file.
There will also be an audio bar at the top of the screen which allows you to play the FilmStart.mp3 file.


To edit the content displayed in the application, open the PSD file in Photoshop.
The background layer is locked, however you can edit most of the text fields - email, film, director, year,
by clicking on the relevant text boxes.
Then save the PSD file over the existing save file to store the changes.
Finally use save as to save the alterations as a jpeg, over the existing FilmScreen.jpeg file in the folder
